# 排期管理接口文档

## 1. 获取排期列表接口

### 请求信息
- **URL**: `/api/schedules`
- **Method**: `GET`
- **Description**: 获取所有排期列表，支持按影片ID筛选

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| movieId | Long | 否 | 影片ID，不传则返回所有排期 |
| cinemaId | Long | 否 | 影院ID，不传则返回所有排期 |

## 3. 添加排期接口

### 请求信息
- **URL**: `/api/schedules`
- **Method**: `POST`
- **Description**: 添加新排期

### 请求体
```json
{
  "movieId": 1,
  "cinemaId": 1,
  "hallName": "3号厅",
  "startTime": "2025-03-24 19:30:00",
  "endTime": "2025-03-24 21:58:00",
  "price": 50.00,
  "status": 1
}
```

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| movieId | Long | 是 | 影片ID |
| cinemaId | Long | 是 | 影院ID |
| hallName | String | 是 | 影厅名称 |
| startTime | String | 是 | 开始时间（格式：yyyy-MM-dd HH:mm:ss） |
| endTime | String | 是 | 结束时间（格式：yyyy-MM-dd HH:mm:ss） |
| price | Double | 是 | 票价 |
| status | Integer | 否 | 状态，默认1（1-可用，0-不可用） |

### 响应数据
```json
{
  "code": 200,
  "message": "添加成功",
  "data": {
    "id": 3,
    "movieId": 1,
    "movieName": "流浪地球3",
    "cinemaId": 1,
    "cinemaName": "万达影城",
    "hallName": "3号厅",
    "startTime": "2025-03-24 19:30:00",
    "endTime": "2025-03-24 21:58:00",
    "price": 50.00,
    "status": 1,
    "createTime": "2025-03-20 11:00:00",
    "updateTime": "2025-03-20 11:00:00"
  }
}
```

## 5. 删除排期接口

### 请求信息
- **URL**: `/api/schedules/{id}`
- **Method**: `DELETE`
- **Description**: 删除排期

### 路径参数
| 参数名 | 类型 | 说明 |
|--------|------|------|
| id | Long | 排期ID |

### 响应数据
```json
{
  "code": 200,
  "message": "删除成功",
  "data": null
}
```

## 6. 接口错误码

| 错误码 | 描述 |
|--------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未授权 |
| 404 | 排期不存在 |
| 500 | 服务器内部错误 |

## 7. 数据库表结构

### schedule 表

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PRIMARY KEY, AUTO_INCREMENT | 排期ID |
| movie_id | BIGINT | NOT NULL, FOREIGN KEY | 影片ID |
| cinema_id | BIGINT | NOT NULL, FOREIGN KEY | 影院ID |
| hall_name | VARCHAR(50) | NOT NULL | 影厅名称 |
| start_time | DATETIME | NOT NULL | 开始时间 |
| end_time | DATETIME | NOT NULL | 结束时间 |
| price | DECIMAL(10,2) | NOT NULL | 票价 |
| status | INT | DEFAULT 1 | 状态：1-可用，0-不可用 |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |
