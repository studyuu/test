# 用户认证接口文档

## 1. 用户登录接口

### 请求信息
- **URL**: `/api/auth/login`
- **Method**: `POST`
- **Description**: 用户登录系统

### 请求体
```json
{
  "username": "admin",
  "password": "123456"
}
```

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| username | String | 是 | 用户名 |
| password | String | 是 | 密码 |

### 响应数据

**成功响应 (code=200)**
```json
{
  "code": 200,
  "message": "登录成功",
  "data": {
    "id": 1,
    "userId": 1,
    "username": "admin",
    "nickname": "管理员",
    "email": "admin@example.com",
    "phone": "13800138000",
    "avatar": "https://picsum.photos/100/100?random=20",
    "role": "admin"
  }
}
```

**失败响应 (code=401)**
```json
{
  "code": 401,
  "message": "密码错误",
  "data": null
}
```

## 2. 用户注册接口

### 请求信息
- **URL**: `/api/auth/register`
- **Method**: `POST`
- **Description**: 新用户注册

### 请求体
```json
{
  "username": "newuser",
  "password": "password123",
  "nickname": "新用户",
  "phone": "13800138003"
}
```

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| username | String | 是 | 用户名 |
| password | String | 是 | 密码 |
| nickname | String | 否 | 昵称，默认为用户名 |
| phone | String | 否 | 手机号 |

### 响应数据

**成功响应 (code=200)**
```json
{
  "code": 200,
  "message": "注册成功",
  "data": {
    "id": 3,
    "userId": 3,
    "username": "newuser",
    "nickname": "新用户",
    "email": null,
    "phone": "13800138003",
    "avatar": null,
    "role": "user"
  }
}
```

**失败响应 (code=409)**
```json
{
  "code": 409,
  "message": "用户名已存在",
  "data": null
}
```

## 3. 用户退出登录接口

### 请求信息
- **URL**: `/api/auth/logout`
- **Method**: `POST`
- **Description**: 用户退出登录

### 响应数据
```json
{
  "code": 200,
  "message": "退出成功",
  "data": null
}
```

## 4. 获取当前用户信息接口

### 请求信息
- **URL**: `/api/auth/current-user`
- **Method**: `GET`
- **Description**: 获取当前登录用户信息

### 请求参数
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| username | String | 是 | 用户名 |

### 响应数据
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "id": 1,
    "userId": 1,
    "username": "admin",
    "nickname": "管理员",
    "email": "admin@example.com",
    "phone": "13800138000",
    "avatar": "https://picsum.photos/100/100?random=20",
    "role": "admin"
  }
}
```

## 5. 接口错误码

| 错误码 | 描述 |
|--------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未授权（密码错误/用户不存在/用户已禁用） |
| 404 | 用户不存在 |
| 409 | 用户名已存在 |
| 500 | 服务器内部错误 |

## 6. 登录流程图

```
用户输入用户名密码
      ↓
调用 POST /api/auth/login
      ↓
验证用户是否存在
      ↓
验证用户状态（是否被禁用）
      ↓
验证密码是否正确
      ↓
返回用户信息（包含role字段）
      ↓
前端存储token和用户信息
      ↓
根据role跳转到对应页面（用户首页/后台管理）
```
