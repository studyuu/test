# 控制台接口文档

## 1. 获取统计数据接口

### 请求信息
- **URL**: `/api/dashboard/stats`
- **Method**: `GET`
- **Description**: 获取控制台首页的统计数据

### 响应数据
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "todayOrders": 156,
    "todayRevenue": 12580.00,
    "totalUsers": 3256,
    "totalMovies": 48
  }
}
```

### 响应字段说明
| 字段名 | 类型 | 说明 |
|--------|------|------|
| todayOrders | Integer | 今日订单数 |
| todayRevenue | Double | 今日营收（元） |
| totalUsers | Integer | 注册用户总数 |
| totalMovies | Integer | 影片总数 |

---

## 2. 获取订单趋势数据接口

### 请求信息
- **URL**: `/api/dashboard/order-trend`
- **Method**: `GET`
- **Description**: 获取最近7天的订单趋势数据

### 响应数据
```json
{
  "code": 200,
  "message": "success",
  "data": [
    {"day": "周一", "count": 120, "revenue": 9600},
    {"day": "周二", "count": 150, "revenue": 12000},
    {"day": "周三", "count": 130, "revenue": 10400},
    {"day": "周四", "count": 180, "revenue": 14400},
    {"day": "周五", "count": 160, "revenue": 12800},
    {"day": "周六", "count": 140, "revenue": 11200},
    {"day": "周日", "count": 156, "revenue": 12480}
  ]
}
```

### 响应字段说明
| 字段名 | 类型 | 说明 |
|--------|------|------|
| day | String | 星期几 |
| count | Integer | 订单数量 |
| revenue | Double | 营收金额 |

---

## 3. 获取影片类型分布接口

### 请求信息
- **URL**: `/api/dashboard/movie-type-distribution`
- **Method**: `GET`
- **Description**: 获取影片类型分布数据

### 响应数据
```json
{
  "code": 200,
  "message": "success",
  "data": [
    {"type": "科幻", "count": 15, "percentage": 30},
    {"type": "喜剧", "count": 12, "percentage": 25},
    {"type": "动作", "count": 10, "percentage": 20},
    {"type": "爱情", "count": 8, "percentage": 15},
    {"type": "其他", "count": 5, "percentage": 10}
  ]
}
```

### 响应字段说明
| 字段名 | 类型 | 说明 |
|--------|------|------|
| type | String | 影片类型 |
| count | Integer | 该类型影片数量 |
| percentage | Integer | 占比（%） |

---

## 4. 获取最近订单列表接口

### 请求信息
- **URL**: `/api/dashboard/recent-orders`
- **Method**: `GET`
- **Description**: 获取最近的订单列表

### 请求参数
| 参数名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| limit | Integer | 否 | 10 | 返回数量 |

### 响应数据
```json
{
  "code": 200,
  "message": "success",
  "data": [
    {
      "orderId": "ORD202403240001",
      "movieTitle": "流浪地球3",
      "cinemaName": "万达影城（CBD店）",
      "hallName": "IMAX厅",
      "showTime": "2024-03-24 14:30:00",
      "seats": "A1,A2",
      "totalPrice": 130.00,
      "status": "已完成",
      "createTime": "2024-03-24 10:30:00"
    },
    {
      "orderId": "ORD202403240002",
      "movieTitle": "热辣滚烫",
      "cinemaName": "CGV影城（颐堤港店）",
      "hallName": "巨幕厅",
      "showTime": "2024-03-24 16:00:00",

      "totalPrice": 96.00,
      "status": "待支付",

```

### 响应字段说明
| 字段名 | 类型 | 说明 |
|--------|------|------|
| orderId | String | 订单号 |
| movieTitle | String | 影片名称 |
| cinemaName | String | 影院名称 |
| hallName | String | 影厅名称 |
| showTime | String | 放映时间 |
| seats | String | 座位号（逗号分隔） |
| totalPrice | Double | 订单金额 |
| status | String | 订单状态（待支付、已完成、已取消） |
| createTime | String | 创建时间 |

## 5. 获取订单详情接口

### 请求信息
- **URL**: `/api/dashboard/order-detail`
- **Method**: `GET`
- **Description**: 根据订单ID获取订单详情

### 请求参数
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| orderId | String | 是 | 订单号 |

### 响应数据
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "orderId": "ORD202403240001",
    "movieTitle": "流浪地球3",
    "cinemaName": "万达影城（CBD店）",
    "hallName": "IMAX厅",
    "showTime": "2024-03-24 14:30:00",
    "seats": "[{\"row\":1,\"rowLabel\":\"A\",\"col\":1,\"colLabel\":\"1\"}]",
    "totalPrice": 130.00,
    "status": "已完成",
    "userName": "张三",
    "userPhone": "138****8888",
    "createTime": "2024-03-24 10:30:00"
  }
}
```

### 响应字段说明
| 字段名 | 类型 | 说明 |
|--------|------|------|
| orderId | String | 订单号 |
| movieTitle | String | 影片名称 |
| cinemaName | String | 影院名称 |
| hallName | String | 影厅名称 |
| showTime | String | 放映时间 |
| seats | String | 座位信息（JSON格式） |
| totalPrice | Double | 订单金额 |
| status | String | 订单状态（待支付、已完成、已取消） |
| userName | String | 下单用户姓名 |
| userPhone | String | 下单用户手机号（脱敏） |
| createTime | String | 创建时间 |
