# 影片页面接口文档

## 1. 获取影片列表（带筛选）

**接口地址**: `GET /api/movies`

**请求参数**:

| 参数名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| type | String | 否 | 电影类型：全部/爱情/喜剧/动作/科幻/动画/剧情/悬疑/恐怖 |
| region | String | 否 | 地区：全部/中国大陆/美国/韩国/日本/中国香港/中国台湾/其他 |
| year | String | 否 | 年代：全部/2025/2024/2023/2022/2021/2020/更早 |
| status | Integer | 否 | 状态：1-上映中，2-即将上映，不传则返回全部 |
| page | Integer | 否 | 页码，默认1 |
| size | Integer | 否 | 每页数量，默认10 |

**响应示例**:

```json
{
  "code": 200,
  "message": "查询成功",
  "data": {
    "content": [
      {
        "movieId": 1,
        "title": "流浪地球3",
        "poster": "https://picsum.photos/300/450?random=1",
        "category": "科幻/冒险",
        "region": "中国大陆",
        "duration": 148,
        "director": "郭帆",
        "actors": "吴京, 刘德华, 李雪健, 沙溢",
        "releaseDate": "2025-02-10",
        "status": 1,
        "rating": 9.2,
        "englishTitle": "The Wandering Earth III",
        "price": 45
      }
    ],
    "totalElements": 6,
    "totalPages": 1,
    "size": 10,
    "number": 0
  }
}
```

## 2. 获取电影类型列表

**接口地址**: `GET /api/movies/types`

**响应示例**:

```json
{
  "code": 200,
  "message": "查询成功",
  "data": [
    "爱情", "喜剧", "动作", "科幻", "动画", "剧情", "悬疑", "恐怖"
  ]
}
```

---

## 4. 获取年代列表

```json
  "data": ["2025", "2024", "2023", "2022", "2021", "2020", "更早"]
}
```

---

## 5. 获取热映电影

**接口地址**: `GET /api/movies/hot`

**说明**: 获取正在上映的电影（status=1），按评分排序

**响应示例**:

```json
{
  "code": 200,
  "message": "查询成功",
  "data": [
    {
      "movieId": 1,
      "title": "流浪地球3",
      "poster": "https://picsum.photos/300/450?random=1",
      "category": "科幻/冒险",
      "rating": 9.2,
      "releaseDate": "2025-02-10"
    }
  ]
}
```

---

## 6. 获取即将上映电影

**接口地址**: `GET /api/movies/coming`

**说明**: 获取即将上映的电影（status=2）

**响应示例**:

```json
{
  "code": 200,
  "message": "查询成功",
  "data": [
    {
      "movieId": 6,
      "title": "沙丘2",
      "poster": "https://picsum.photos/300/450?random=6",
      "category": "科幻/动作",
      "releaseDate": "2025-03-08"
    }
  ]
}
```

## 7. 根据ID获取影片详情

**接口地址**: `GET /api/movies/{id}`

**路径参数**:
