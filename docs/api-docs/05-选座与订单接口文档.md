## 1. 获取排期详情（含座位信息）接口

- **URL**: `/api/schedules/{id}/detail`
- **Method**: `GET`

### 路径参数
| 参数名 | 类型 | 说明 |
|--------|------|------|
| id | Long | 排期ID |

{
  "code": 200,

  "data": {
    "schedule": {

## 2. 创建订单接口

### 请求信息
- **URL**: `/api/orders`
- **Method**: `POST`
- **Description**: 创建电影票订单

### 请求体
```json
{
  "scheduleId": 1,
  "userId": 1,
  "selectedSeats": [
    {
      "row": 1,
      "rowLabel": "A",
      "col": 1,
      "colLabel": "1"
    }
  ]
}
```

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| scheduleId | Long | 是 | 排期ID |
| userId | Long | 是 | 用户ID |
| selectedSeats | Array | 是 | 已选座位列表 |

### 响应数据
```json
{
  "code": 200,
  "message": "创建订单成功",
  "data": {
    "orderId": "ORD1711111111111",
    "movieName": "流浪地球3",
    "cinemaName": "万达影城",
    "hallName": "IMAX厅",
    "startTime": "2025-03-24 14:30:00",
    "selectedSeats": [
      {
        "rowLabel": "A",
        "colLabel": "1"
      }
    ],
    "totalPrice": 65.00,
    "status": "pending",
    "createTime": "2025-03-20 12:00:00"
  }
}
```

## 4. 获取用户订单列表接口

### 请求信息
- **URL**: `/api/users/{userId}/orders`
- **Method**: `GET`
- **Description**: 获取用户的订单列表（支持分页）

### 路径参数
| 参数名 | 类型 | 说明 |
|--------|------|------|
| userId | Long | 用户ID |

### 请求参数
| 参数名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| pageNum | Integer | 否 | 1 | 页码 |
| pageSize | Integer | 否 | 10 | 每页数量 |
| status | String | 否 | all | 状态筛选（all-全部，pending-待支付，completed-已完成，cancelled-已取消） |

### 响应数据
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "list": [
      {
        "orderId": "ORD1711111111111",
        "movieId": 1,
        "movieTitle": "流浪地球3",
        "poster": "https://example.com/poster.jpg",
        "cinemaName": "万达影城（CBD店）",
        "hallName": "IMAX厅",
        "showTime": "2025-03-24 14:30:00",
        "seats": "[{\"row\":5,\"rowLabel\":\"5\",\"col\":6,\"colLabel\":\"6\"},{\"row\":5,\"rowLabel\":\"5\",\"col\":7,\"colLabel\":\"7\"}]",
        "totalPrice": 130.00,
        "status": "已完成",
        "createTime": "2025-03-23 10:30:00"
      }
    ],
    "total": 10
  }
}
```

### 响应字段说明
| 字段名 | 类型 | 说明 |
|--------|------|------|
| orderId | String | 订单号 |
| movieId | Integer | 影片ID |
| movieTitle | String | 影片名称 |
| poster | String | 影片海报URL |
| cinemaName | String | 影院名称 |
| hallName | String | 影厅名称 |
| showTime | String | 放映时间 |
| seats | String | 座位信息（JSON格式） |
| totalPrice | Double | 订单金额 |
| status | String | 订单状态（待支付、已完成、已取消） |
| createTime | String | 创建时间 |
| total | Integer | 总记录数 |

### 7.4 申请退票

#### 请求信息
- **URL**: `/api/orders/{orderId}/refund`
- **Method**: `POST`
- **Description**: 用户申请退票（仅已完成的订单可申请）

#### 路径参数
| 参数名 | 类型 | 说明 |
|--------|------|------|
| orderId | String | 订单号 |

#### 请求体
```json
{
  "reason": "personal"
}
```

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| reason | String | 是 | 退票原因（personal-个人原因，schedule_change-场次时间变更，service_issue-影院服务问题，other-其他原因） |

#### 响应数据
```json
{
  "code": 200,
  "message": "退票申请已提交，等待审核",
  "data": "refunding"
}
```

## 8. 数据库表结构

### order 表
| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PRIMARY KEY, AUTO_INCREMENT | 订单ID |
| order_id | VARCHAR(50) | NOT NULL, UNIQUE | 订单号 |
| user_id | BIGINT | NOT NULL, FOREIGN KEY | 用户ID |
| schedule_id | BIGINT | NOT NULL, FOREIGN KEY | 排期ID |
| seats | TEXT | NOT NULL | 座位信息（JSON格式） |
| total_price | DECIMAL(10,2) | NOT NULL | 总价 |
| status | VARCHAR(20) | DEFAULT 'pending' | 状态：pending-待支付，paid-已支付，cancelled-已取消 |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

### seat 表
| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PRIMARY KEY, AUTO_INCREMENT | 座位ID |
| schedule_id | BIGINT | NOT NULL, FOREIGN KEY | 排期ID |
| row_num | INT | NOT NULL | 行号 |
| row_label | VARCHAR(10) | NOT NULL | 行标签（如A,B,C） |
| col_num | INT | NOT NULL | 列号 |
| col_label | VARCHAR(10) | NOT NULL | 列标签（如1,2,3） |
| status | VARCHAR(20) | DEFAULT 'available' | 状态：available-可用，sold-已售 |
| order_id | VARCHAR(50) | DEFAULT NULL | 关联订单ID |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |
