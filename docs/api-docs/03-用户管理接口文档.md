## 3. 添加用户接口

### 请求信息
- **URL**: `/api/users`
- **Method**: `POST`
- **Description**: 添加新用户

### 请求体
```json
{
  "username": "newuser",
  "password": "password123",
  "nickname": "新用户",
  "email": "newuser@example.com",
  "phone": "13800138003",
  "avatar": "https://picsum.photos/100/100?random=23",
  "role": "user"
}
```

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| username | String | 是 | 用户名 |
| password | String | 是 | 密码 |
| nickname | String | 是 | 昵称 |
| email | String | 否 | 邮箱 |
| phone | String | 否 | 手机号 |
| avatar | String | 否 | 头像URL |
| role | String | 否 | 角色，默认user |

### 响应数据
```json
{
  "code": 200,
  "message": "添加成功",
  "data": {
    "id": 3,
    "username": "newuser",
    "nickname": "新用户",
    "email": "newuser@example.com",
    "phone": "13800138003",
    "avatar": "https://picsum.photos/100/100?random=23",
    "role": "user",
    "status": "active",
    "createTime": "2025-01-03 16:00:00",
    "updateTime": "2025-01-03 16:00:00"
  }
}
```

## 5. 禁用/启用用户接口

### 请求信息
- **URL**: `/api/users/{id}/status`
- **Method**: `PUT`
- **Description**: 切换用户状态（禁用/启用）

### 路径参数
| 参数名 | 类型 | 说明 |
|--------|------|------|
| id | Long | 用户ID |

### 请求体
```json
{
  "status": "disabled"
}
```

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| status | String | 是 | 状态：active（启用）或 disabled（禁用） |

### 响应数据
```json
{
  "code": 200,
  "message": "状态更新成功",
  "data": {
    "id": 2,
    "username": "user1",
    "nickname": "张三",
    "status": "disabled"
  }
}
```

## 7. 接口错误码

| 错误码 | 描述 |
|--------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未授权 |
| 404 | 用户不存在 |
| 409 | 用户名已存在 |
| 500 | 服务器内部错误 |

## 8. 数据库表结构

### sys_user 表

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PRIMARY KEY, AUTO_INCREMENT | 用户ID |
| username | VARCHAR(50) | NOT NULL, UNIQUE | 用户名 |
| password | VARCHAR(100) | NOT NULL | 密码（加密存储） |
| nickname | VARCHAR(50) | NOT NULL | 昵称 |
| email | VARCHAR(100) | NULL | 邮箱 |
| phone | VARCHAR(20) | NULL | 手机号 |
| avatar | VARCHAR(255) | NULL | 头像URL |
| role | VARCHAR(20) | DEFAULT 'user' | 角色：admin/user |
| status | VARCHAR(20) | DEFAULT 'active' | 状态：active/disabled |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |
